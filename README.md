### 用户注册接口

 1. 接口示例：http://127.0.0.1/entrance/rest.html?method=member.register&login=v&pwd=1&email=1@qq.com&phone=18703111111
 2. 接口参数：
 
| 参数名   | 是否必填 | 说明                           |
| -------- | -------- | ------------------------------ |
| login    | 必填     | 用户名                         |
| pwd      | 必填     | 用户密码                       |
| email    | 选填     | 用户邮箱                       |
| phone    | 选填     | 用户手机号                     |
| nickName | 选填     | 用户昵称                       |
| realName | 选填     | 真实姓名                       |
| regType  | 选填     | 注册类型0：个人1：企业 默认为0 |
 3. 接口返回：

| 状态码 | 说明           |
| ------ | -------------- |
| 1000   | 注册成功       |
| 1001   | 用户名已经存在 |
